# Getting started

This page helps you start using BiciRadar quickly.

## 1) Install the app

- iPhone: install from App Store.
- Android: install from Play Store or internal beta link (if shared with you).
- Apple Watch: install from the paired iPhone Watch app.

## 2) Grant permissions

For the best experience, allow:

- Location permission (preferably "While using the app").
- Notifications (optional, but recommended).

If location is denied, some features may use a fallback city-center location and be less precise.

## 3) Pick your city and favorites

- Open the app and select your city bike network.
- Mark your most-used stations as favorites (for example Home and Work).

## 4) Try voice commands

- iPhone: use Siri shortcuts from the iPhone page.
- Apple Watch: use Siri shortcuts from the watch page.
- Android: use Google Assistant intents and phrases from the Android page.
